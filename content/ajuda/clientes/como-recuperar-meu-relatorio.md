# Acesse a página de segunda via

Use a página oficial:

[Recuperar meu relatório](https://api.canvaspiritual.com/segunda-via.html)

## Como buscar

1. abra a página de segunda via;
2. informe o e-mail utilizado no pagamento;
3. clique em **Buscar**;
4. localize o relatório desejado;
5. clique em **Baixar PDF**.

A busca apresenta os relatórios pagos localizados dentro do período informado pela própria página.

## Nenhum relatório apareceu

Confira:

- se o e-mail foi digitado corretamente;
- se você utilizou outro e-mail na compra;
- se o pagamento foi confirmado;
- se o relatório já foi concluído;
- se a compra está dentro do período pesquisado.

Caso ainda não encontre, fale com o suporte e informe o e-mail usado no pagamento.
